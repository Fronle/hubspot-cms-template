# Fronle, LLC HubSpot CMS Template

This is a template for HubSpot CMS. 

# Languages
- SCSS
- Vue.JS
- Typescript
- HUBL

# Getting Started
- `yarn install`
- Fill in variables.scss with your theming from Bootstrap
- Seting hubspot.config.yml

# How to Run
- `yarn install`
- `yarn watch:dev`
- `hs watch dist {folder_name}`
